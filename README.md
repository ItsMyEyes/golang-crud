# golang-crud

# Route
```
GET    /api/v1/home/          
GET    /api/v1/todo/          
POST   /api/v1/todo/          
GET    /api/v1/todo/:id       
DELETE /api/v1/todo/:id       
POST   /api/v1/user/register  
POST   /api/v1/user/login     
GET    /api/v1/user/logout    
GET    /api/v1/               
```

  git clone https://github.com/ItsMyEyes/golang-crud //repo git
  code golang-crud // open vsc studio
